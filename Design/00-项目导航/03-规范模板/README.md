## 📝 文档格式规范摘要

### 文档头部

```markdown
# 文档标题

> **创建日期**：YYYY-MM-DD
> **最后更新**：YYYY-MM-DD
> **版本**：1.0
> **状态**：草稿/进行中/完成

---
```

### 目录结构

- 使用H2（##）作为主要章节标题
- 使用H3（###）作为子章节标题
- 每个章节之间使用`---`分隔

### 代码块

- 使用语法高亮：\`\`\`python, \`\`\`sql, \`\`\`yaml等
- 代码块前后留空行
- 长代码添加注释说明

### 表格

- 使用标准Markdown表格语法
- 表头加粗
- 适当对齐
