# Design项目文档格式规范

> **创建日期**：2025-12-01
> **最后更新**：2025-12-01
> **版本**：1.0
> **状态**：完成 ✅

---

## 📋 概述

本规范定义了Design项目中所有文档的格式标准，确保文档一致性和可维护性。

---

## 1. 文档头部规范

### 1.1 标准头部格式

```markdown
# 文档标题

> **创建日期**：YYYY-MM-DD
> **最后更新**：YYYY-MM-DD
> **版本**：X.Y
> **状态**：草稿/进行中/完成 ✅

---
```

### 1.2 状态标记

| 状态 | 图标 | 说明 |
|------|------|------|
| 草稿 | 📝 | 初始创建，内容不完整 |
| 进行中 | 🔄 | 正在编写或修订 |
| 完成 | ✅ | 内容完整，可供使用 |
| 待审核 | 👀 | 等待评审 |
| 归档 | 📦 | 历史版本，不再更新 |

---

## 2. 章节结构规范

### 2.1 标题层级

```markdown
# H1 - 文档标题（每个文档只有一个）
## H2 - 主要章节
### H3 - 子章节
#### H4 - 详细内容（尽量避免使用H5及以下）
```

### 2.2 章节分隔

- 主要章节（H2）之间使用`---`分隔
- 子章节（H3）之间不需要分隔线
- 文档末尾添加分隔线和更新信息

### 2.3 目录编号

对于长文档，使用编号：

```markdown
## 1. 第一章
### 1.1 第一节
### 1.2 第二节
## 2. 第二章
```

---

## 3. 内容格式规范

### 3.1 列表格式

**无序列表**：

```markdown
- 第一项
- 第二项
  - 子项
  - 子项
```

**有序列表**：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

**任务列表**（仅用于进度跟踪文档）：

```markdown
- [x] 已完成任务
- [ ] 待完成任务
```

### 3.2 强调格式

| 格式 | 语法 | 用途 |
|------|------|------|
| **粗体** | `**文本**` | 重要概念、关键词 |
| *斜体* | `*文本*` | 术语、引用 |
| `代码` | `` `代码` `` | 内联代码、文件名 |
| ~~删除~~ | `~~文本~~` | 废弃内容 |

### 3.3 引用格式

**普通引用**：

```markdown
> 引用内容
```

**多行引用**：

```markdown
> 第一行
> 第二行
```

**嵌套引用**：

```markdown
> 外层引用
>> 内层引用
```

## 4. 代码块规范

### 4.1 语言标识

| 语言 | 标识 |
|------|------|
| Python | `python` |
| SQL | `sql` |
| JavaScript | `javascript` |
| TypeScript | `typescript` |
| YAML | `yaml` |
| JSON | `json` |
| TLA+ | `tla` |
| Shell | `bash` |
| Markdown | `markdown` |
| 纯文本 | `text` |

### 4.2 代码块格式

```markdown
```python
def example_function():
    """示例函数"""
    return "Hello, World!"
```
```

### 4.3 代码注释

- 关键代码添加注释
- 复杂逻辑添加说明
- 示例代码添加输出结果

## 5. 表格规范

### 5.1 基本格式

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 值1 | 值2 | 值3 |
| 值4 | 值5 | 值6 |
```

### 5.2 对齐方式

```markdown
| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| 内容 | 内容 | 内容 |
```

### 5.3 表格说明

复杂表格前添加说明文字，表格后可添加注释。

## 6. 链接规范

### 6.1 内部链接

```markdown
[文档标题](./相对路径/文件名.md)
[章节链接](./文件名.md#章节锚点)
```

### 6.2 外部链接

```markdown
[链接文本](https://example.com)
```

### 6.3 图片链接

```markdown
![图片说明](./images/image.png)
```

---

## 7. 特殊元素规范

### 7.1 提示框

```markdown
> **注意**：重要提示内容

> **警告**：警告内容

> **提示**：提示内容
```

### 7.2 数学公式

**内联公式**：`$E = mc^2$`

**块级公式**：

```markdown
$$
E = mc^2
$$
```

### 7.3 脚注

```markdown
这是正文内容[^1]。

[^1]: 这是脚注内容。
```
