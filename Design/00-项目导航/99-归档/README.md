## 📋 目录说明

本目录用于存放已完成或不再活跃的历史文档。

---

## 📁 归档内容

### 历史计划文档

- 已完成的项目计划
- 已废弃的方案设计
- 历史版本的规范文档

### 历史报告文档

- 早期的进度报告
- 已关闭的任务清单
- 完成的里程碑文档

---

## 📝 归档规则

1. **归档条件**：
   - 文档内容已过时
   - 任务已完成且不再需要更新
   - 被新版本替代的文档

2. **归档流程**：
   - 在文档头部标记为"归档"状态
   - 移动到99-归档目录
   - 更新相关索引文档

3. **命名规范**：
   - 保留原文件名
   - 可添加归档日期后缀：`原文件名-归档-YYYY-MM-DD.md`
