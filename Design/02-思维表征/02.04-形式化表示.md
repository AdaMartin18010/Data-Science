## 2. 集合论符号

### 2.1. 基本集合符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `∈` | 属于 | 元素属于集合 | `a ∈ A` |
| `∉` | 不属于 | 元素不属于集合 | `a ∉ A` |
| `⊆` | 子集 | A是B的子集 | `A ⊆ B` |
| `⊂` | 真子集 | A是B的真子集 | `A ⊂ B` |
| `∪` | 并集 | A和B的并集 | `A ∪ B` |
| `∩` | 交集 | A和B的交集 | `A ∩ B` |
| `\` | 差集 | A减去B | `A \ B` |
| `×` | 笛卡尔积 | A和B的笛卡尔积 | `A × B` |
| `\|A\|` | 基数 | 集合A的元素个数 | `\|A\| = n` |
| `∅` | 空集 | 空集合 | `∅` |
| `P(A)` | 幂集 | A的所有子集 | `P(A)` |

### 2.2. 集合操作

**关系定义**：

```text
R ⊆ dom(A₁) × dom(A₂) × ... × dom(Aₙ)
```

**元组表示**：

```text
t = (a₁, a₂, ..., aₙ) ∈ R
```

**属性值**：

```text
t[A] = aᵢ  (如果 A = Aᵢ)
```

## 3. 逻辑符号

### 3.1. 命题逻辑符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `∧` | 合取 | 且 | `P ∧ Q` |
| `∨` | 析取 | 或 | `P ∨ Q` |
| `¬` | 否定 | 非 | `¬P` |
| `→` | 蕴含 | 如果...则 | `P → Q` |
| `↔` | 等价 | 当且仅当 | `P ↔ Q` |
| `⊤` | 真 | 真值 | `⊤` |
| `⊥` | 假 | 假值 | `⊥` |

### 3.2. 谓词逻辑符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `∀` | 全称量词 | 对所有 | `∀x P(x)` |
| `∃` | 存在量词 | 存在 | `∃x P(x)` |
| `∃!` | 唯一存在 | 存在唯一 | `∃!x P(x)` |
| `:` | 使得 | 使得 | `{x : P(x)}` |
| `\|` | 使得 | 使得 | `{x \| P(x)}` |

### 3.3. 逻辑表达式示例

**函数依赖定义**：

```text
X → Y ⟺ ∀ t₁, t₂ ∈ R, t₁[X] = t₂[X] ⟹ t₁[Y] = t₂[Y]
```

**查询包含定义**：

```text
Q₁ ⊇ Q₂ ⟺ ∀ I, Q₁(I) ⊇ Q₂(I)
```

## 4. 关系代数符号

### 4.1. 基本操作符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `σ_θ(R)` | 选择 | 选择满足条件θ的元组 | `σ_{age>25}(Students)` |
| `π_A(R)` | 投影 | 投影到属性集合A | `π_{name,age}(Students)` |
| `R ∪ S` | 并 | R和S的并 | `R ∪ S` |
| `R - S` | 差 | R减去S | `R - S` |
| `R × S` | 笛卡尔积 | R和S的笛卡尔积 | `R × S` |
| `R ⋈_θ S` | θ连接 | 条件θ的连接 | `R ⋈_{R.A=S.B} S` |
| `R ⋈ S` | 自然连接 | 自然连接 | `R ⋈ S` |
| `R ÷ S` | 除 | R除以S | `R ÷ S` |
| `ρ_{A→B}(R)` | 重命名 | 属性重命名 | `ρ_{name→fullname}(R)` |

### 4.2. 关系代数表达式

**复杂查询**：

```text
π_{name, title}(
    σ_{grade='A'}(
        Students ⋈ Enrollments ⋈ Courses
    )
)
```

**等价变换**：

```text
σ_θ(R ⋈ S) ≡ σ_θ(R) ⋈ S  (如果θ只涉及R的属性)
```

## 5. 范畴论符号

### 5.1. 基本范畴符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `C` | 范畴 | 范畴C | `C` |
| `Ob(C)` | 对象集合 | C的对象集合 | `Ob(C)` |
| `Hom_C(A,B)` | 态射集合 | 从A到B的态射 | `Hom_C(A,B)` |
| `f: A → B` | 态射 | 从A到B的态射 | `f: A → B` |
| `g ∘ f` | 复合 | 态射的复合 | `g ∘ f` |
| `id_A` | 恒等态射 | A的恒等态射 | `id_A` |

### 5.2. 函子符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `F: C → D` | 函子 | 从C到D的函子 | `F: C → D` |
| `F(A)` | 对象映射 | A在F下的像 | `F(A)` |
| `F(f)` | 态射映射 | f在F下的像 | `F(f)` |

### 5.3. 自然变换符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `η: F → G` | 自然变换 | 从F到G的自然变换 | `η: F → G` |
| `η_A` | 分量 | A处的分量 | `η_A: F(A) → G(A)` |

### 5.4. 范畴论表达式

**函子性质**：

```text
F(g ∘ f) = F(g) ∘ F(f)
F(id_A) = id_{F(A)}
```

**自然性条件**：

```text
η_B ∘ F(f) = G(f) ∘ η_A
```

## 6. 函数依赖符号

### 6.1. 函数依赖符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `X → Y` | 函数依赖 | X函数决定Y | `{id} → {name, age}` |
| `X →→ Y` | 多值依赖 | X多值决定Y | `{course} →→ {teacher}` |
| `F` | 函数依赖集合 | 函数依赖集合 | `F = {X → Y, Y → Z}` |
| `F⁺` | 闭包 | F的闭包 | `F⁺` |
| `X⁺` | 属性闭包 | X的闭包 | `X⁺` |
| `F ⊢ X → Y` | 推导 | 从F可推导X→Y | `F ⊢ X → Y` |
| `F ⊨ X → Y` | 语义蕴含 | F语义蕴含X→Y | `F ⊨ X → Y` |

### 6.2. 函数依赖表达式

**Armstrong公理**：

```text
自反性：Y ⊆ X ⟹ X → Y
增广性：X → Y ⟹ XZ → YZ
传递性：X → Y ∧ Y → Z ⟹ X → Z
```

**闭包定义**：

```text
X⁺ = {A | F ⊢ X → A}
F⁺ = {X → Y | F ⊢ X → Y}
```

## 7. 图论符号

### 7.1. 图符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `G = (V, E)` | 图 | 顶点集合V和边集合E | `G = (V, E)` |
| `V(G)` | 顶点集合 | 图G的顶点集合 | `V(G)` |
| `E(G)` | 边集合 | 图G的边集合 | `E(G)` |
| `(u, v)` | 边 | 从u到v的边 | `(u, v) ∈ E` |
| `deg(v)` | 度 | 顶点v的度 | `deg(v)` |
| `d(u, v)` | 距离 | u到v的距离 | `d(u, v)` |
| `P = (v₁, v₂, ..., vₙ)` | 路径 | 路径P | `P` |

### 7.2. 知识图谱符号

| 符号 | 名称 | 含义 | 示例 |
|------|------|------|------|
| `KG = (E, R, T)` | 知识图谱 | 实体、关系、三元组 | `KG` |
| `(e₁, r, e₂)` | 三元组 | 实体e₁通过r关联e₂ | `(e₁, r, e₂)` |
| `E` | 实体集合 | 实体集合 | `E` |
| `R` | 关系集合 | 关系集合 | `R` |
| `T` | 三元组集合 | 三元组集合 | `T` |

## 8. 符号索引

### 8.1. 按类别索引

**集合论**：

- `∈, ∉, ⊆, ⊂, ∪, ∩, \, ×, |A|, ∅, P(A)`

**逻辑**：

- `∧, ∨, ¬, →, ↔, ∀, ∃, ∃!, :, |`

**关系代数**：

- `σ, π, ∪, -, ×, ⋈, ÷, ρ`

**范畴论**：

- `C, Ob(C), Hom_C, →, ∘, id, F, η`

**函数依赖**：

- `→, →→, F, F⁺, X⁺, ⊢, ⊨`

**图论**：

- `G = (V, E), V(G), E(G), (u, v), deg, d, P`

### 8.2. 按字母顺序索引

| 符号 | 类别 | 含义 |
|------|------|------|
| `→` | 逻辑/函数依赖 | 蕴含/函数依赖 |
| `↔` | 逻辑 | 等价 |
| `×` | 集合论/关系代数 | 笛卡尔积 |
| `∘` | 范畴论 | 复合 |
| `÷` | 关系代数 | 除 |
| `∈` | 集合论 | 属于 |
| `∉` | 集合论 | 不属于 |
| `⊆` | 集合论 | 子集 |
| `⊂` | 集合论 | 真子集 |
| `∪` | 集合论/关系代数 | 并 |
| `∩` | 集合论 | 交 |
| `\` | 集合论 | 差 |
| `∧` | 逻辑 | 合取 |
| `∨` | 逻辑 | 析取 |
| `¬` | 逻辑 | 否定 |
| `∀` | 逻辑 | 全称量词 |
| `∃` | 逻辑 | 存在量词 |
| `σ` | 关系代数 | 选择 |
| `π` | 关系代数 | 投影 |
| `⋈` | 关系代数 | 连接 |
| `ρ` | 关系代数 | 重命名 |

## 9. 使用指南

### 9.1. 符号使用原则

1. **一致性**：在整个文档中使用一致的符号
2. **清晰性**：优先使用标准符号
3. **上下文**：在上下文中明确符号含义
4. **定义**：首次使用时给出定义

### 9.2. LaTeX渲染

**Markdown中的LaTeX**：

```latex
行内公式：$X \to Y$
块级公式：
$$
\forall t_1, t_2 \in R, t_1[X] = t_2[X] \implies t_1[Y] = t_2[Y]
$$
```

### 9.3. Unicode符号

**常用Unicode符号**：

- `→` (U+2192): 箭头
- `∀` (U+2200): 全称量词
- `∃` (U+2203): 存在量词
- `∈` (U+2208): 属于
- `∉` (U+2209): 不属于
- `⊆` (U+2286): 子集
- `×` (U+00D7): 乘号
- `∘` (U+2218): 复合
