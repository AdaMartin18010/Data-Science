## 一、分类概述

### 1.1 分类维度

```text
┌─────────────────────────────────────────────────────────────────────────┐
│                        SQL概念分类维度                                   │
└─────────────────────────────────────────────────────────────────────────┘

                           SQL概念
                              │
        ┌─────────┬───────────┼───────────┬─────────┐
        │         │           │           │         │
        ▼         ▼           ▼           ▼         ▼
   抽象层次    功能类型     演进阶段    复杂度     应用域
        │         │           │           │         │
    ┌───┴───┐ ┌───┴───┐  ┌───┴───┐  ┌───┴───┐ ┌───┴───┐
    数学    │ 定义   │  SQL-86  │  基础   │ OLTP  │
    理论    │ 操作   │  SQL-92  │  进阶   │ OLAP  │
    语言    │ 查询   │  SQL:99  │  高级   │ 混合  │
    实现    │ 控制   │  SQL:23  │  专家   │       │
```

### 1.2 分类原则

| 原则 | 说明 |
|------|------|
| **完整性** | 覆盖所有SQL核心概念 |
| **互斥性** | 同一维度内分类不重叠 |
| **层次性** | 支持多级细分 |
| **实用性** | 便于学习和查找 |

## 二、按抽象层次分类

### 2.1 层次结构

```text
Level 0: 数学基础
─────────────────────────────────────────────────────────────
├── 集合论
│   ├── 集合 (Set)
│   ├── 子集 (Subset)
│   ├── 笛卡尔积 (Cartesian Product)
│   └── 幂集 (Power Set)
│
├── 函数论
│   ├── 函数 (Function)
│   ├── 映射 (Mapping)
│   ├── 单射/满射/双射
│   └── 函数复合
│
└── 谓词逻辑
    ├── 命题 (Proposition)
    ├── 谓词 (Predicate)
    ├── 量词 (Quantifier)
    └── 推理规则

Level 1: 理论模型
─────────────────────────────────────────────────────────────
├── 关系模型
│   ├── 关系 (Relation)
│   ├── 属性 (Attribute)
│   ├── 元组 (Tuple)
│   ├── 域 (Domain)
│   └── 关系模式 (Schema)
│
├── 依赖理论
│   ├── 函数依赖 (FD)
│   ├── 多值依赖 (MVD)
│   ├── 连接依赖 (JD)
│   └── Armstrong公理
│
├── 范式理论
│   ├── 1NF/2NF/3NF
│   ├── BCNF
│   ├── 4NF/5NF
│   └── 分解算法
│
└── 关系代数
    ├── 选择 (σ)
    ├── 投影 (π)
    ├── 连接 (⋈)
    ├── 集合操作 (∪∩-)
    └── 聚合 (γ)

Level 2: 语言层
─────────────────────────────────────────────────────────────
├── DDL (数据定义)
│   ├── CREATE
│   ├── ALTER
│   ├── DROP
│   └── TRUNCATE
│
├── DML (数据操作)
│   ├── INSERT
│   ├── UPDATE
│   ├── DELETE
│   └── MERGE
│
├── DQL (数据查询)
│   ├── SELECT
│   ├── FROM/JOIN
│   ├── WHERE
│   ├── GROUP BY/HAVING
│   ├── ORDER BY
│   └── LIMIT/OFFSET
│
├── DCL (数据控制)
│   ├── GRANT
│   ├── REVOKE
│   └── DENY
│
└── TCL (事务控制)
    ├── BEGIN
    ├── COMMIT
    ├── ROLLBACK
    └── SAVEPOINT

Level 3: 实现层
─────────────────────────────────────────────────────────────
├── 存储引擎
│   ├── 页面 (Page)
│   ├── 缓冲池 (Buffer Pool)
│   ├── 索引结构 (B+Tree/Hash)
│   └── 日志系统 (WAL)
│
├── 查询处理
│   ├── 解析器 (Parser)
│   ├── 优化器 (Optimizer)
│   ├── 执行器 (Executor)
│   └── 算子 (Operator)
│
├── 事务管理
│   ├── 锁管理 (Lock)
│   ├── MVCC
│   ├── 日志恢复
│   └── 并发控制
│
└── 分布式
    ├── 分区 (Partition)
    ├── 复制 (Replication)
    ├── 一致性协议
    └── 分布式事务
```

### 2.2 层次间关系

```text
依赖链：
数学基础 ──[定义]──→ 理论模型 ──[实现]──→ 语言层 ──[执行]──→ 实现层

示例：
集合论 ──→ 关系 ──→ SELECT ──→ TableScan算子
笛卡尔积 ──→ 连接 ──→ JOIN ──→ HashJoin算子
函数 ──→ 函数依赖 ──→ 约束 ──→ 约束检查
```

## 四、按演进阶段分类

### 4.1 SQL标准演进分类

```text
SQL-86/87 (基础)
─────────────────────────────────────────────────────────────
概念集合:
├── SELECT/FROM/WHERE
├── INSERT/UPDATE/DELETE
├── CREATE TABLE
├── PRIMARY KEY
└── 基本数据类型

SQL-89 (修订)
─────────────────────────────────────────────────────────────
新增概念:
├── 外键约束
├── CHECK约束
└── 事务完整性

SQL-92 (标准化)
─────────────────────────────────────────────────────────────
新增概念:
├── JOIN语法 (INNER/LEFT/RIGHT/FULL)
├── CASE表达式
├── CAST类型转换
├── 字符串操作
├── 日期时间类型
├── 临时表
├── 域约束
└── 入口级/中级/完整级

SQL:1999 (面向对象)
─────────────────────────────────────────────────────────────
新增概念:
├── 用户定义类型
├── 递归查询 (WITH RECURSIVE)
├── 触发器
├── 存储过程
├── 角色
├── 布尔类型
├── 正则表达式
└── OLAP扩展

SQL:2003 (XML和窗口)
─────────────────────────────────────────────────────────────
新增概念:
├── 窗口函数 (OVER/PARTITION BY)
├── 序列 (SEQUENCE)
├── 自动生成列 (IDENTITY)
├── MERGE语句
├── XML支持
└── BIGINT类型

SQL:2008 (增强)
─────────────────────────────────────────────────────────────
新增概念:
├── TRUNCATE
├── FETCH FIRST
├── 增强CASE
└── 数组操作增强

SQL:2011 (时态)
─────────────────────────────────────────────────────────────
新增概念:
├── 系统版本表
├── 应用时间表
├── 时态查询
└── PERIOD定义

SQL:2016 (JSON)
─────────────────────────────────────────────────────────────
新增概念:
├── JSON数据类型
├── JSON路径表达式
├── JSON函数
├── JSON表函数
└── 行模式匹配

SQL:2023 (图和向量)
─────────────────────────────────────────────────────────────
新增概念:
├── 属性图查询 (SQL/PGQ)
├── 多维数组
├── JSON增强
└── 向量支持（草案）
```

### 4.2 演进关系图

```text
SQL-86 ───┬──→ SQL-89 ───┬──→ SQL-92 ───┬──→ SQL:1999
          │              │              │
          │              │              ├── 递归查询
          │              │              ├── 触发器
          │              │              └── 用户类型
          │              │
          │              ├── JOIN语法
          │              └── CASE表达式
          │
          └── 外键
                              │
                              ▼
          SQL:2003 ───┬──→ SQL:2008 ───┬──→ SQL:2011
                      │                │
                      ├── 窗口函数      ├── TRUNCATE
                      └── MERGE         └── FETCH
                              │
                              ▼
          SQL:2016 ────────→ SQL:2023
              │                  │
              └── JSON           ├── SQL/PGQ
                                 └── 向量(草案)
```

## 五、按复杂度分类

### 5.1 学习路径分级

```text
Level 1: 基础 (入门1-2周)
─────────────────────────────────────────────────────────────
必备概念:
├── SELECT基础 (列选择、条件过滤、排序)
├── INSERT/UPDATE/DELETE基础
├── CREATE TABLE基础
├── 主键/外键概念
├── 基本数据类型
└── NULL处理

Level 2: 进阶 (掌握1-2月)
─────────────────────────────────────────────────────────────
核心概念:
├── JOIN类型 (INNER/LEFT/RIGHT/FULL/CROSS)
├── 子查询 (标量/列/表)
├── GROUP BY/HAVING
├── 聚合函数 (SUM/COUNT/AVG/MAX/MIN)
├── 索引基础
├── 事务基础 (COMMIT/ROLLBACK)
└── 约束类型

Level 3: 高级 (精通2-6月)
─────────────────────────────────────────────────────────────
高级概念:
├── 窗口函数 (ROW_NUMBER/RANK/LEAD/LAG)
├── CTE和递归查询
├── 隔离级别
├── 执行计划分析
├── 索引优化
├── 查询重写
├── JSON支持
└── 分区表

Level 4: 专家 (深入6月+)
─────────────────────────────────────────────────────────────
专家概念:
├── 关系代数理论
├── 范式理论
├── 查询优化器原理
├── 存储引擎原理
├── MVCC实现
├── 分布式事务
├── 复制协议
└── 向量数据库
```
