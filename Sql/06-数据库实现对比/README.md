## 概述

本模块对比主流数据库系统（PostgreSQL、MySQL、SQLite、SQL Server）对SQL标准的实现情况。

**核心价值**：

- 理解不同数据库的标准兼容性差异
- 指导跨数据库迁移和兼容性设计
- 支持数据库选型决策
