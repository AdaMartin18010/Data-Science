    │
    └─ 部署环境？
        ├─ 嵌入式/移动 → SQLite
        ├─ 本地服务器 → PostgreSQL/MySQL
        └─ 云环境 → RDS/Aurora

决策树2: 索引选择
═══════════════════════════════════════════════════════════

问题: 创建什么索引？
    │
    ├─ 查询模式？
    │   ├─ 等值查询 → B-Tree索引
    │   ├─ 范围查询 → B-Tree索引
    │   ├─ 前缀匹配 → B-Tree索引
    │   ├─ 全文搜索 → GIN/FTS索引
    │   └─ 地理位置 → GiST索引
    │
    ├─ 列的选择性？
    │   ├─ 高选择性 (>50% unique) → 单列索引
    │   ├─ 低选择性 (<10% unique) → 部分索引
    │   └─ 中等选择性 → 视情况
    │
    ├─ 多列查询？
    │   ├─ AND条件 → 复合索引
    │   ├─ OR条件 → 多个单列索引
    │   └─ 列顺序 → 等值在前，范围在后
    │
    └─ 写入频率？
        ├─ 读多写少 → 多个索引
        ├─ 读写均衡 → 精选索引
        └─ 写多读少 → 最小化索引

权衡矩阵:
索引类型 | 查询性能 | 写入性能 | 空间占用 | 适用场景
--------|---------|---------|---------|--------

Hash    | ⭐⭐⭐⭐⭐| ⭐⭐    | ⭐⭐⭐⭐ | 等值查询
GIN     | ⭐⭐⭐⭐⭐| ⭐      | ⭐⭐    | 全文/数组
GiST    | ⭐⭐⭐⭐ | ⭐⭐    | ⭐⭐⭐   | 地理/范围

### 任务19.1: 术语统一规范

```text
创建文档: Sql/09-整合资源/09.13-SQL术语标准化词典.md

术语分类:
1. 核心概念术语
   ├─ 关系（Relation）vs 表（Table）
   │  统一使用: 理论用"关系"，实践用"表"
   ├─ 元组（Tuple）vs 行（Row）
   │  统一使用: 理论用"元组"，实践用"行"
   ├─ 属性（Attribute）vs 列（Column）
   │  统一使用: 理论用"属性"，实践用"列"
   └─ 域（Domain）vs 数据类型（Data Type）
      统一使用: 理论用"域"，实践用"数据类型"

2. 中英文对照
   概念 | 英文 | 标准中文 | 避免使用
   -----|------|---------|--------
   Transaction | Transaction | 事务 | 交易
   Concurrency | Concurrency | 并发 | 并行
   Isolation | Isolation | 隔离 | 孤立
   Deadlock | Deadlock | 死锁 | 僵局

3. 缩写规范
   缩写 | 全称 | 中文 | 使用场景
   -----|------|------|--------
   ACID | Atomicity, Consistency, Isolation, Durability | 原子性、一致性、隔离性、持久性 | 事务特性
   MVCC | Multi-Version Concurrency Control | 多版本并发控制 | 并发控制
   2PL | Two-Phase Locking | 两阶段锁 | 并发控制
   WAL | Write-Ahead Logging | 预写日志 | 持久化机制
```
