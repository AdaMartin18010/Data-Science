## 概述

本模块涵盖SQL和关系数据库的**理论基础**，包括关系模型、关系代数、关系演算、SQL完备性理论和事务理论。这些理论是理解SQL语言和数据库系统的根基。

**2025增强**：新增完整的数学基础到SQL的因果推导链，以及范式理论的形式化证明。

## 因果关系链

### 1. 数学基础 → 关系模型

```text
ZFC集合论
    │
    ├──→ 笛卡尔积定义 ───→ 关系定义 ───→ 元组/属性
    │
    ├──→ 函数定义 ───→ 函数依赖 ───→ 范式理论
    │
    └──→ 谓词逻辑 ───→ 关系演算 ───→ SQL WHERE
```

### 2. 关系代数 → SQL

```text
关系代数操作              SQL语句
────────────────────────────────────
σ (选择)         ───→    WHERE
π (投影)         ───→    SELECT
× (笛卡尔积)     ───→    CROSS JOIN
⋈ (连接)         ───→    JOIN
∪ (并)           ───→    UNION
∩ (交)           ───→    INTERSECT
− (差)           ───→    EXCEPT
```

### 3. 函数依赖 → 范式

```text
函数依赖 X → Y
    │
    ├──→ Armstrong公理（自反/增广/传递）
    │         │
    │         └──→ 1NF: 原子性
    │                  │
    │                  └──→ 2NF: 完全函数依赖
    │                            │
    │                            └──→ 3NF: 无传递依赖
    │                                      │
    │                                      └──→ BCNF: 决定因素为候选键
    │
    └──→ 多值依赖 ───→ 4NF
              │
              └──→ 连接依赖 ───→ 5NF
```
