## 📋 示例说明

虽然SQLite原生不支持列存储，但可以通过设计模式模拟列存储的优势。本示例展示了三种列存储模拟方案：

1. **垂直分区（列分离）** - 将宽表拆分为多个窄表
2. **列式物化视图** - 为分析查询创建列存储结构
3. **列压缩存储** - 对低基数列应用压缩算法
