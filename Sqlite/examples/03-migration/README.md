## 🎯 使用说明

### 运行示例

```bash
# 基础迁移
python 01-简单表迁移.py

# 复杂迁移
python 02-复杂表迁移.py

# 批量迁移
python 03-批量迁移示例.py
```

### 配置要求

1. **SQLite数据库**：准备测试数据库
2. **PostgreSQL**：配置连接信息
3. **Python依赖**：
   - `sqlite3` (内置)
   - `psycopg2` (需要安装)

### 安装依赖

```bash
pip install psycopg2-binary
```
