## 🚀 快速开始

> 💡 **新手指南**：查看 [快速开始指南（QUICK-START.md）](./QUICK-START.md) 获取更详细的导航和学习路径

### 初学者

1. 阅读 [主文档（readme.md）](./readme.md) 了解完整知识体系
2. 学习 [01-核心架构](./01-核心架构/) 理解SQLite工作原理
3. 实践 [08-编程实践](./08-编程实践/) 掌握基本操作
4. 参考 [examples/](./examples/) 动手实践
5. （可选）深入学习 [11-理论模型](./11-理论模型/) 🆕 和 [12-设计模型](./12-设计模型/) 🆕 理解理论基础

### 开发者

1. 学习 [02-数据模型](./02-数据模型/) 掌握表设计
2. 深入 [08-编程实践](./08-编程实践/) 学习最佳实践
3. 参考 [04-应用场景](./04-应用场景/) 学习真实案例
4. 优化 [03-性能优化](./03-性能优化/) 提升性能
5. 理解 [11-理论模型](./11-理论模型/) 🆕 掌握算法复杂度和性能理论

### 架构师

1. 理解 [01-核心架构](./01-核心架构/) 深入架构设计
2. 分析 [05-对比选型](./05-对比选型/) 做出选型决策
3. 参考 [04-应用场景](./04-应用场景/) 设计架构方案
4. 学习 [12-设计模型](./12-设计模型/) 🆕 掌握设计原则和设计模式
5. 研究 [11-理论模型](./11-理论模型/) 🆕 理解理论基础和系统理论
