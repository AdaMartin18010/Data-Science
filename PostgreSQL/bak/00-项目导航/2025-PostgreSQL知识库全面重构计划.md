## 2. PostgreSQL 17新特性

### 2.1 核心新特性（必须覆盖）

#### 性能增强

**1. 增量备份支持**

```sql
-- 基础全量备份
pg_basebackup -D /backup/full -F tar -z

-- 增量备份（新特性）
pg_basebackup -D /backup/incremental \
  --incremental=/backup/full/backup_manifest \
  -F tar -z
```
