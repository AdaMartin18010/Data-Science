# 10.1.1 Rust基础理论深化

## 理论基础

### Rust语言概述

Rust是一种系统编程语言，专注于内存安全和并发安全：

- **内存安全**：通过所有权系统防止内存泄漏和数据竞争
- **零成本抽象**：高级抽象不带来运行时开销
- **并发安全**：编译时保证线程安全
- **性能优先**：接近C/C++的性能表现

### 核心特性

- **所有权系统**：编译时内存管理
- **借用检查**：防止数据竞争
- **生命周期**：管理引用的有效性
- **模式匹配**：强大的控制流工具

## 所有权系统

### 所有权规则

```rust
// 所有权基本概念
fn main() {
    // 1. 每个值都有一个所有者
    let s1 = String::from("hello");

    // 2. 同一时间只能有一个所有者
    let s2 = s1; // s1的所有权移动到s2，s1不再有效
    // println!("{}", s1); // 编译错误：s1已经被移动

    // 3. 当所有者离开作用域，值被丢弃
    {
        let s3 = String::from("world");
        // s3在这里有效
    } // s3离开作用域，被丢弃

    // 4. 函数参数的所有权
    let s4 = String::from("test");
    takes_ownership(s4); // s4的所有权移动到函数
    // println!("{}", s4); // 编译错误：s4已经被移动

    // 5. 返回值转移所有权
    let s5 = gives_ownership(); // 函数返回值获得所有权
    println!("{}", s5);
}

fn takes_ownership(some_string: String) {
    println!("{}", some_string);
} // some_string离开作用域，被丢弃

fn gives_ownership() -> String {
    let some_string = String::from("hello");
    some_string // 返回some_string，所有权转移给调用者
}
```

### 借用和引用

```rust
// 借用系统
fn main() {
    let mut s1 = String::from("hello");

    // 不可变借用
    let len = calculate_length(&s1);
    println!("Length of '{}' is {}.", s1, len);

    // 可变借用
    change(&mut s1);
    println!("Changed string: {}", s1);

    // 借用规则示例
    let mut s2 = String::from("hello");

    let r1 = &s2; // 不可变借用
    let r2 = &s2; // 另一个不可变借用
    // let r3 = &mut s2; // 编译错误：不能同时有可变和不可变借用

    println!("{} and {}", r1, r2);
    // r1和r2在这里不再使用

    let r3 = &mut s2; // 现在可以创建可变借用
    println!("{}", r3);
}

fn calculate_length(s: &String) -> usize {
    s.len()
} // s离开作用域，但因为它不拥有数据，所以什么都不会发生

fn change(some_string: &mut String) {
    some_string.push_str(", world");
}
```

```rust
    for (i, &item) in bytes.iter().enumerate() {

            return &s[0..i];
        }
    }

    &s[..]
}
```

## 生命周期

### 生命周期注解

```rust
// 生命周期基础
fn main() {
    let string1 = String::from("abcd");
    let string2 = "xyz";

    let result = longest(string1.as_str(), string2);
    println!("The longest string is {}", result);
}

// 生命周期注解
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() {
        x
    } else {
        y
    }
}

// 结构体中的生命周期
struct ImportantExcerpt<'a> {
    part: &'a str,
}

fn main() {
    let novel = String::from("Call me Ishmael. Some years ago...");
    let first_sentence = novel.split('.').next().unwrap();
    let i = ImportantExcerpt {
        part: first_sentence,
    };
    println!("{}", i.part);
}

// 生命周期省略规则
fn first_word(s: &str) -> &str {
    // 编译器自动推断生命周期
    let bytes = s.as_bytes();

    for (i, &item) in bytes.iter().enumerate() {
        if item == b' ' {
            return &s[0..i];
        }
    }

    &s[..]
}

// 静态生命周期
fn main() {
    let s: &'static str = "I have a static lifetime.";
    println!("{}", s);
}
```

## 错误处理

### Result类型

```rust
// Result基础用法
use std::fs::File;
use std::io::{self, Read};

fn read_username_from_file() -> Result<String, io::Error> {
    let mut f = File::open("hello.txt")?;
    let mut s = String::new();
    f.read_to_string(&mut s)?;
    Ok(s)
}

// 自定义错误类型
#[derive(Debug)]
enum CustomError {
    IoError(std::io::Error),
    ParseError(std::num::ParseIntError),
}

impl From<std::io::Error> for CustomError {
    fn from(err: std::io::Error) -> CustomError {
        CustomError::IoError(err)
    }
}

impl From<std::num::ParseIntError> for CustomError {
    fn from(err: std::num::ParseIntError) -> CustomError {
        CustomError::ParseError(err)
    }
}

fn read_and_parse() -> Result<i32, CustomError> {
    let mut s = String::new();
    std::io::stdin().read_line(&mut s)?;
    let num: i32 = s.trim().parse()?;
    Ok(num)
}

// 错误处理模式
fn handle_errors() {
    // 使用match
    let result = read_and_parse();
    match result {
        Ok(num) => println!("Parsed number: {}", num),
        Err(e) => println!("Error: {:?}", e),
    }

    // 使用unwrap
    let num = read_and_parse().unwrap();
    println!("Number: {}", num);

    // 使用expect
    let num = read_and_parse().expect("Failed to parse number");
    println!("Number: {}", num);

    // 使用?操作符
    fn process_data() -> Result<(), CustomError> {
        let num = read_and_parse()?;
        println!("Processing number: {}", num);
        Ok(())
    }
}
```

### Option类型

```rust
// Option基础用法
fn find_item(items: &[i32], target: i32) -> Option<usize> {
    for (index, &item) in items.iter().enumerate() {
        if item == target {
            return Some(index);
        }
    }
    None
}

fn main() {
    let items = vec![1, 2, 3, 4, 5];

    // 使用match
    match find_item(&items, 3) {
        Some(index) => println!("Found at index: {}", index),
        None => println!("Item not found"),
    }

    // 使用if let
    if let Some(index) = find_item(&items, 3) {
        println!("Found at index: {}", index);
    } else {
        println!("Item not found");
    }

    // 使用unwrap
    let index = find_item(&items, 3).unwrap();
    println!("Found at index: {}", index);

    // 使用unwrap_or
    let index = find_item(&items, 10).unwrap_or(0);
    println!("Index: {}", index);
}

// Option高级用法
fn divide(a: f64, b: f64) -> Option<f64> {
    if b == 0.0 {
        None
    } else {
        Some(a / b)
    }
}

fn main() {
    let result = divide(10.0, 2.0);
    println!("Result: {:?}", result);

    let result = divide(10.0, 0.0);
    println!("Result: {:?}", result);

    // 使用map
    let result = divide(10.0, 2.0).map(|x| x * 2.0);
    println!("Doubled result: {:?}", result);

    // 使用and_then
    let result = divide(10.0, 2.0).and_then(|x| divide(x, 2.0));
    println!("Chained result: {:?}", result);
}
```

## 并发编程

### 线程

```rust
use std::thread;
use std::time::Duration;

fn main() {
    // 基本线程创建
    let handle = thread::spawn(|| {
        for i in 1..10 {
            println!("hi number {} from the spawned thread!", i);
            thread::sleep(Duration::from_millis(1));
        }
    });

    for i in 1..5 {
        println!("hi number {} from the main thread!", i);
        thread::sleep(Duration::from_millis(1));
    }

    handle.join().unwrap();

    // 使用move闭包
    let v = vec![1, 2, 3];

    let handle = thread::spawn(move || {
        println!("Here's a vector: {:?}", v);
    });

    handle.join().unwrap();
}
```

### 消息传递

```rust
use std::sync::mpsc;
use std::thread;

fn main() {
    let (tx, rx) = mpsc::channel();

    thread::spawn(move || {
        let val = String::from("hi");
        tx.send(val).unwrap();
        // println!("val is {}", val); // 编译错误：val已经被发送
    });

    let received = rx.recv().unwrap();
    println!("Got: {}", received);

    // 多个发送者
    let (tx, rx) = mpsc::channel();

    let tx1 = tx.clone();
    thread::spawn(move || {
        let vals = vec![
            String::from("hi"),
            String::from("from"),
            String::from("the"),
            String::from("thread"),
        ];

        for val in vals {
            tx1.send(val).unwrap();
            thread::sleep(Duration::from_millis(1));
        }
    });

    thread::spawn(move || {
        let vals = vec![
            String::from("more"),
            String::from("messages"),
            String::from("for"),
            String::from("you"),
        ];

        for val in vals {
            tx.send(val).unwrap();
            thread::sleep(Duration::from_millis(1));
        }
    });

    for received in rx {
        println!("Got: {}", received);
    }
}
```

### 共享状态

```rust
use std::sync::{Arc, Mutex};
use std::thread;

fn main() {
    // 使用Mutex
    let counter = Arc::new(Mutex::new(0));
    let mut handles = vec![];

    for _ in 0..10 {
        let counter = Arc::clone(&counter);
        let handle = thread::spawn(move || {
            let mut num = counter.lock().unwrap();
            *num += 1;
        });
        handles.push(handle);
    }

    for handle in handles {
        handle.join().unwrap();
    }

    println!("Result: {}", *counter.lock().unwrap());

    // 使用RwLock
    use std::sync::RwLock;

    let data = Arc::new(RwLock::new(vec![1, 2, 3, 4, 5]));
    let mut handles = vec![];

    // 读取线程
    for i in 0..3 {
        let data = Arc::clone(&data);
        let handle = thread::spawn(move || {
            let data = data.read().unwrap();
            println!("Reader {}: {:?}", i, *data);
        });
        handles.push(handle);
    }

    // 写入线程
    let data = Arc::clone(&data);
    let handle = thread::spawn(move || {
        let mut data = data.write().unwrap();
        data.push(6);
        println!("Writer: {:?}", *data);
    });
    handles.push(handle);

    for handle in handles {
        handle.join().unwrap();
    }
}
```

## 智能指针

### Box

```rust
// Box智能指针
fn main() {
    // 在堆上存储数据
    let b = Box::new(5);
    println!("b = {}", b);

    // 递归数据结构
    #[derive(Debug)]
    enum List {
        Cons(i32, Box<List>),
        Nil,
    }

    let list = List::Cons(1, Box::new(List::Cons(2, Box::new(List::Nil))));
    println!("{:?}", list);

    // 实现Deref trait
    use std::ops::Deref;

    struct MyBox<T>(T);

    impl<T> MyBox<T> {
        fn new(x: T) -> MyBox<T> {
            MyBox(x)
        }
    }

    impl<T> Deref for MyBox<T> {
        type Target = T;

        fn deref(&self) -> &Self::Target {
            &self.0
        }
    }

    let x = 5;
    let y = MyBox::new(x);

    assert_eq!(5, x);
    assert_eq!(5, *y);
}
```

### Rc

```rust
// Rc智能指针
use std::rc::Rc;

fn main() {
    // 共享所有权
    let a = Rc::new(List::Cons(5, Rc::new(List::Cons(10, Rc::new(List::Nil)))));
    println!("count after creating a = {}", Rc::strong_count(&a));

    let b = List::Cons(3, Rc::clone(&a));
    println!("count after creating b = {}", Rc::strong_count(&a));

    {
        let c = List::Cons(4, Rc::clone(&a));
        println!("count after creating c = {}", Rc::strong_count(&a));
    }
    println!("count after c goes out of scope = {}", Rc::strong_count(&a));

    #[derive(Debug)]
    enum List {
        Cons(i32, Rc<List>),
        Nil,
    }
}

// 弱引用
use std::rc::{Rc, Weak};
use std::cell::RefCell;

#[derive(Debug)]
struct Node {
    value: i32,
    parent: RefCell<Weak<Node>>,
    children: RefCell<Vec<Rc<Node>>>,
}

fn main() {
    let leaf = Rc::new(Node {
        value: 3,
        parent: RefCell::new(Weak::new()),
        children: RefCell::new(vec![]),
    });

    println!("leaf parent = {:?}", leaf.parent.borrow().upgrade());

    let branch = Rc::new(Node {
        value: 5,
        parent: RefCell::new(Weak::new()),
        children: RefCell::new(vec![Rc::clone(&leaf)]),
    });

    *leaf.parent.borrow_mut() = Rc::downgrade(&branch);

    println!("leaf parent = {:?}", leaf.parent.borrow().upgrade());
}
```

## 模式匹配

### 基本模式

```rust
// 模式匹配基础
fn main() {
    let x = 1;

    match x {
        1 => println!("one"),
        2 => println!("two"),
        3 => println!("three"),
        _ => println!("anything"),
    }

    // 绑定值
    let x = Some(5);
    let y = 10;

    match x {
        Some(50) => println!("Got 50"),
        Some(y) => println!("Matched, y = {:?}", y),
        _ => println!("Default case, x = {:?}", x),
    }

    println!("at the end: x = {:?}, y = {:?}", x, y);

    // 多重模式
    let x = 1;

    match x {
        1 | 2 | 3 => println!("one, two, or three"),
        _ => println!("anything"),
    }

    // 范围模式
    let x = 5;

    match x {
        1..=5 => println!("one through five"),
        _ => println!("something else"),
    }

    // 解构
    let p = Point { x: 0, y: 7 };

    match p {
        Point { x, y: 0 } => println!("On the x axis at {}", x),
        Point { x: 0, y } => println!("On the y axis at {}", y),
        Point { x, y } => println!("On neither axis: ({}, {})", x, y),
    }
}

struct Point {
    x: i32,
    y: i32,
}
```

### 高级模式

```rust
// 高级模式匹配
fn main() {
    // 守卫
    let num = Some(4);

    match num {
        Some(x) if x < 5 => println!("less than five: {}", x),
        Some(x) => println!("{}", x),
        None => (),

    let msg = Message::Hello { id: 5 };

        }
        Message::Hello { id: 10..=12 } => {
            println!("Found an id in another range")
        }
        Message::Hello { id } => println!("Found some other id: {}", id),

    let s = Some(String::from("Hello!"));

    if let Some(_) = s {
        println!("found a string");
    }

    let origin = Point { x: 0, y: 0 };

    match origin {
        Point { x, .. } => println!("x is {}", x),
    }

    // 条件忽略
    let numbers = (2, 4, 8, 16, 32);

    match numbers {
        (first, _, third, _, fifth) => {
            println!("Some numbers: {}, {}, {}", first, third, fifth)
        }
    }
}
```

## 最佳实践

### 性能优化

1. **零成本抽象**：使用高级抽象不带来性能损失
2. **内存布局**：合理设计数据结构的内存布局
3. **迭代器**：使用迭代器而不是手动循环
4. **编译优化**：利用编译器的优化能力

### *错误处理*

1. **Result类型**：使用Result处理可恢复错误
2. **Option类型**：使用Option处理可选值
3. **自定义错误**：为特定领域定义错误类型
4. **错误传播**：使用?操作符传播错误

### 并发安全

1. **消息传递**：优先使用消息传递而非共享状态
2. **智能指针**：使用Arc和Mutex管理共享状态
3. **生命周期**：确保引用的生命周期正确
4. **数据竞争**：编译时防止数据竞争

### 代码组织

1. **模块系统**：合理组织代码模块
2. **trait系统**：使用trait定义接口
3. **泛型编程**：使用泛型提高代码复用性
4. **文档注释**：为公共API编写文档

## 总结

Rust基础理论为构建安全、高效的系统软件提供了强大的工具。通过所有权系统、生命周期管理、并发编程和模式匹配等特性，可以编写出既安全又高效的代码。
