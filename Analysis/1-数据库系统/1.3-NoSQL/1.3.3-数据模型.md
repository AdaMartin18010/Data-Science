# 1.3.3 数据模型

## 1. 数据结构设计

- 键值、文档、列族、图等多种结构。
- 典型设计：用户文档、订单集合、图节点与边。

## 2. 索引与查询

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 主键索引 | 唯一性、加速定位 | 键值/文档/列存 |
| 二级索引 | 支持多字段查询 | 文档/列存 |
| 复合索引 | 多条件组合 | 复杂查询 |
| 图索引   | 节点/边关系 | 图数据库 |
