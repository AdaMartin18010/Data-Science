# 1.1.98 数据库JSON数据模型：JSON查询与模式验证的形式化（纯形式化）

## 目录

1. JSON模型
2. JSON查询
3. 模式验证

## 1. 模型

- JSON值：null、boolean、number、string、array、object；
- JSON路径：点记法、方括号记法、通配符。

## 2. 查询

- JSONPath：基于路径的查询语言；
- 定理：JSONPath查询的复杂度为O(|文档| × |路径|)；
- 命题：JSON查询可转换为关系代数表达式。

## 3. 验证

- JSON Schema：基于模式的语言；
- 定理：JSON Schema验证的复杂度为O(|文档| × |模式|)；
- 命题：模式组合满足单调性（子模式蕴含父模式）。□
