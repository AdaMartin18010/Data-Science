## 2. 索引与分区

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 主键索引 | 唯一性、加速查询 | 主键字段 |
| 普通索引 | 提高查询效率 | 高频查询 |
| 联合索引 | 多字段组合 | 复合条件 |
| 分区表   | 水平拆分 | 大表分区 |

## 3. 行业案例与多表征

### 3.1. 零售行业：订单表分区设计

```sql
CREATE TABLE orders (
  id INT PRIMARY KEY,
  user_id INT,
  amount DECIMAL(10,2),
  order_date DATE
)
PARTITION BY RANGE (YEAR(order_date)) (
  PARTITION p2022 VALUES LESS THAN (2023),
  PARTITION p2023 VALUES LESS THAN (2024)
);
```
