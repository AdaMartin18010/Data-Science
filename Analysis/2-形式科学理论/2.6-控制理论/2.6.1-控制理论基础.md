# 控制理论基础

## 1. 控制理论概述

控制理论是研究动态系统行为调节和优化的数学理论，广泛应用于工程、生物学、经济学和社会科学等领域。控制理论的核心目标是设计控制器，使系统按照期望的方式运行。

### 1.1 控制系统的基本概念

**定义 1.1.1 (控制系统)**
控制系统是一个四元组 CS = (S, A, T, C)，其中：

- S 是状态空间
- A 是动作空间
- T: S × A → S 是转移函数
- C: S × A → ℝ 是成本函数

**定义 1.1.2 (反馈控制)**
反馈控制器是一个函数 π: S → A，使得系统行为由以下方程描述：

```text
s_{t+1} = T(s_t, π(s_t))
```

### 1.2 控制系统的分类

1. **按系统特性分类**：
   - 线性系统 vs 非线性系统
   - 时变系统 vs 时不变系统
   - 连续时间系统 vs 离散时间系统

2. **按复杂度分层**：
   - 单输入单输出(SISO)系统
   - 多输入多输出(MIMO)系统
   - 分布式系统
   - 网络化系统

## 2. 线性控制系统理论

### 2.1 线性系统表示

**定义 2.1.1 (线性系统)**
线性系统由以下方程描述：

```text
ẋ(t) = Ax(t) + Bu(t)
y(t) = Cx(t) + Du(t)
```

其中：

- x(t) ∈ ℝⁿ 是状态向量
- u(t) ∈ ℝᵐ 是输入向量
- y(t) ∈ ℝᵖ 是输出向量
- A, B, C, D 是系统矩阵

### 2.2 可控性与可观测性

**定义 2.2.1 (可控性)**
线性系统是可控的，当且仅当可控性矩阵：

```text
W = [B AB A²B ... Aⁿ⁻¹B]
```

满秩。

**定理 2.2.1 (可控性判据)**
线性系统可控当且仅当 rank(W) = n。

**定义 2.2.2 (可观测性)**
线性系统是可观测的，当且仅当可观测性矩阵：

```text
O = [C; CA; CA²; ...; CAⁿ⁻¹]
```

满秩。

**定理 2.2.2 (可观测性判据)**
线性系统可观测当且仅当 rank(O) = n。

### 2.3 稳定性分析

**定义 2.3.1 (稳定性)**
线性系统渐近稳定当且仅当所有特征值的实部为负。

**定理 2.3.1 (李雅普诺夫稳定性)**
对于线性系统 ẋ = Ax，以下条件等价：

1. 系统渐近稳定
2. 对于任意正定矩阵 Q，存在唯一正定矩阵 P 满足 A^T P + PA = -Q
3. 所有特征值实部为负

## 3. 现代控制理论

### 3.1 状态反馈控制

**定义 3.1.1 (状态反馈)**
状态反馈控制律：

```text
u(t) = -Kx(t)
```

其中 K 是反馈增益矩阵。

**定理 3.1.1 (极点配置)**
如果线性系统可控，则存在状态反馈增益 K 使闭环系统具有任意指定的特征值。

### 3.2 最优控制

**定义 3.2.1 (最优控制问题)**
最优控制问题是寻找控制输入使得性能指标最小化：

```text
min J = ∫₀ᵀ L(x(t), u(t), t)dt + φ(x(T))
s.t. ẋ(t) = f(x(t), u(t), t)
```

**定理 3.2.1 (LQR控制)**
对于线性系统和二次型性能指标：

```text
J = ∫₀^∞ (x^T Q x + u^T R u) dt
```

最优控制为 u = -Kx，其中 K = R^(-1)B^T P，P 是代数黎卡提方程的解：

```text
A^T P + PA - PBR^(-1)B^T P + Q = 0
```

## 4. 非线性控制理论

### 4.1 李雅普诺夫稳定性理论

**定义 4.1.1 (李雅普诺夫函数)**
李雅普诺夫函数 V: X → ℝ 满足：

1. V(0) = 0
2. V(x) > 0, ∀x ≠ 0
3. V̇(x) ≤ 0

**定理 4.1.1 (李雅普诺夫稳定性)**
如果存在李雅普诺夫函数，则系统在原点稳定。

### 4.2 反馈线性化

**定义 4.2.1 (相对度)**
非线性系统的相对度是输出对输入的导数中第一个显式包含输入的阶数。

**定理 4.2.1 (精确反馈线性化)**
如果非线性系统有良定义的相对度，则存在坐标变换和反馈控制使系统在新坐标下表现为线性系统。

### 4.3 滑模控制

**定义 4.3.1 (滑模面)**
滑模面是状态空间中的超平面：s(x) = 0

**定义 4.3.2 (滑模控制)**
滑模控制律：u = u_{eq} + u_{sw}，其中 u_{eq} 是等效控制，u_{sw} 是开关控制。

## 5. 鲁棒控制理论

### 5.1 不确定性建模

**定义 5.1.1 (参数不确定性)**
系统参数变化：A = A₀ + ΔA，B = B₀ + ΔB

**定义 5.1.2 (结构化不确定性)**
不确定性可表示为结构化块：Δ = diag(δ₁I₁, ..., δₙIₙ)

### 5.2 H∞控制

**定义 5.2.1 (H∞范数)**
系统的H∞范数是其最大奇异值：

```text
||G||∞ = sup_ω σ̄(G(jω))
```

**定理 5.2.1 (H∞控制)**
H∞控制问题是设计控制器 K 使闭环系统的H∞范数小于给定值 γ。

## 6. 智能控制与现代应用

### 6.1 自适应控制

**定义 6.1.1 (自适应控制)**
自适应控制器根据系统参数估计实时调整控制参数。

### 6.2 神经网络控制

**定义 6.2.1 (神经网络控制)**
使用神经网络近似非线性函数，构建控制律。

## 8. 总结与展望

控制理论为动态系统的分析和设计提供了完整的理论框架，从经典控制到现代控制，从线性系统到非线性系统，从确定性控制到鲁棒控制，形成了一个完整的理论体系。随着人工智能、量子计算等新技术的发展，控制理论也在不断拓展其应用边界，为复杂系统的控制提供更加有力的工具。
