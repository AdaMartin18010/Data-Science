## 多表征

数学基础理论分支支持多种表征方式，包括：

- 数学符号、公式、结构
- 数学图谱、结构图
- 向量/张量（数学对象嵌入）
- 自然语言与可视化
这些表征可互映，提升数学理论的表达力。

## 形式化语义

- 语义域：$D$，如数学对象集、结构集、模型空间

- 语义一致性：每个数学结构/公式在$D$中有明确定义
