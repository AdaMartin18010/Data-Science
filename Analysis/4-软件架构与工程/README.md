# 4-软件架构与工程

## 📋 概述

软件架构与工程是软件系统的设计、构建、部署和维护的完整工程体系。本分支涵盖了从基础理论到具体实践，从设计模式到微服务架构，从IoT系统到工作流域的完整软件工程知识体系。
